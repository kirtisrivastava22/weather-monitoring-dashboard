## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To run this project locally, you must create a .env.local file in the root directory and add your OpenWeatherMap API key:
Code snippet

NEXT_PUBLIC_API_KEY=your_api_key_here
